# 现在来给文章添加一张图片吧

比如：

1. 使用标签（typora无法预览）

   ```java
   // {% hello-world 2.jpeg img %}
   ```
   {% asset_img 2.jpeg img %}

3. 直接文件夹\图片名（typora可以预览）

   ```java
   //![img](hello-world\2.jpeg)
   ```

   ![](hello-world\2.jpeg)

4. 直接使用图片名（typora中无法预览）

   ```java
   //![img](2.jpeg)
   ```

   ![img](2.jpeg)
